About Prevalence Analysis

This section explores the prevalence patterns of COVID-19 serological tests across IgM and IgG antibody types. The visualizations include scatter plots showing the relationship between false positive rates (FP) and true positive rates (TP), K-means clustering to identify test groupings, and elbow method analysis to determine optimal cluster counts.

Interactive Features: Filter by test type (ELISA, Lateral Flow, Rapid Tests, etc.), select specific antibody panels (IgM/IgG), and explore individual cluster groups to understand test performance patterns.
